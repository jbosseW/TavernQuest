Collection of modular heads for use with the v3 LPC character bases <https://opengameart.org/content/lpc-character-bases>. Create all kinds of humanoid folk!

The following heads are new to this submission:

- wartotaur
- mouse
- rat
- rabbit
- pig
- sheep
- goblin
- troll
- vampire
- frankenstein
- alien
- mouse child
- rat child
- rabbit child
- pig child
- sheep child
- troll child

Includes sheets for use with all animations included in that submission (cast, thrust, walk, slash, shoot, hurt, jump, run, idle, sit). All adult heads work with all adult bodies (male, female, teen/androgynous, muscular, pregnant); child heads work with child body. 

Includes templates and scripts for generating full spritesheets from a smaller number of images; see `_build/build.sh` (requires `lpctools` <https://github.com/bluecarrot16/lpctools>). 

Could be adapted to work with LPC Revised bodies, but may require different masks and/or offsets.
